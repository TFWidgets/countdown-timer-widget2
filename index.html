🎯 Countdown Widget Demo

Example of using the countdown widget:

<script src="https://countdown-timer-widget2.tf-widgets.com/embed.js" data-id="demo"></script>

📋 How to Use

1. Create a configuration file in the configs/ folder

2. Insert the code into your website:

<script src="https://countdown-timer-widget2.tf-widgets.com/embed.js" data-id="your-client-id"></script>

3. Done! The widget will automatically load with your settings.
